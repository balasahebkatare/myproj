Python is a powerful programming language created by Guido van Rossum, in the early 1990s and is a successor of ABC language.It is a ideal language for Scripting and rapid development of applications.

Django Tutorial >>>>

Django is a web development Framework, it is purely written in Python. It was designed to make Common web-development task fast and easy. It is maintaned by the Django Software Foundation. Django was started as internal project, in 2003 and released in July 2005. Django comes with a lightweigth Web Server which provides development environment to facilate end-to-end application development and testing.

Flask Tutorial >>>>

Flask is a micro framework for python web development. In 2004, some python interested people came together and they formed a group, called Pocoo. Flask was developed at pocco by Armin Ronacher of Pocoo and it is based on Werkzeug and Jinja2. It is purely written in Python language.
