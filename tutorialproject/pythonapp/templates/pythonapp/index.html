Python Tutorial

Python is an easy to learn, powerful programming language. It has efficient high-level data structures and a simple but effective approach to object-oriented programming.

Python is a interpreted language and it can be easilyt extended with new functions and datatypes.

The main intention of desining this tutorial is to make Python language concepts as much as simple with the help of examples, so that anyone can understand easily within a short period of time.

All the examples which are given in this Python Tutorial are tested on Python 3.6.6 and Ubantu 16 LTS Operating System.
