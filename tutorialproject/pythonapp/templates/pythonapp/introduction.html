The main intention of desining this tutorial is to make Python language concepts as much as simple with the help of examples, so that anyone can understand easily within a short period of time. All the examples which are given in this python tutorial are tested on Python 3.6.6 and Ubantu 16.04 LTS operating system.

Python is a powerful programming language created by Guido van Rossum, in the early 1990s and is a successor of ABC language.It is a ideal language for Scripting and rapid development of applications.

Python is simple to use, but it is a real programming language which offers much more structure and support for large programs. Python allows us to split our program into small parts called modules. We can use these modules in another programs.

Python is a interpreted language. There is no need of compilation and linking for a python program hence it saves our time during program development. And also the programs written in python are shorter than C/CPP or JAVA programs. It is Object Oriented as well as Procedure Oriented programming language.

Python Implementations

Besides the Python implementation there are some other popular Python implementations such as,

  • CPython

    • This is the most popular and Original implementation of the Python language which is written in C language.

  • JPython

    • This implementation is written in JAVA language. We can use this implementation as a scripting language with JAVA applications.

  • PyPy

    • This is another implementation of the Python which is purely written in Python language. It provides several advanced features which are not available in other implementations. It provides Just-in-Time compiler which helps to run python programs faster. If we want to run our programs faster then we should use the PyPy implementation.

  • IronPython

    • This implementation is for a .NET. This implementation helps to generate the Intermediate Language Code (IL).
